CASO DE USO: Registro de servicios
1.	El guía selecciona la opción "Registrar Servicios" en el menú del sitio web.
2.	El sistema muestra el formulario de registro de servicios al guía.
3.	El guía ingresa los atributos del servicio (título, descripción, ubicación, precio, fecha y hora, tipo) en el formulario de registro.
4.	El guía envía los atributos del servicio al sistema haciendo clic en el botón "Guardar".
5.	El sistema valida que los datos ingresados estén completos.
6.	Si los datos están incompletos, el sistema muestra un mensaje de error y vuelve a mostrar el formulario de registro de servicios para que el guía pueda corregir los campos incompletos.
7.	Si los datos están completos, el sistema almacena el servicio en una base de datos y muestra un mensaje de confirmación al guía indicando que el servicio ha sido registrado con éxito.
